# 技能上传

上传 ZIP 格式的自定义技能包。每个技能包必须包含 SKILL.md 文件作为技能定义。

## 使用方法

1. 点击上传按钮打开对话框
2. 将 ZIP 文件拖放到上传区域，或点击浏览选择文件
3. 确认选中的文件名和大小
4. 点击"上传"开始上传过程
5. 等待上传完成并查看成功提示
6. 上传成功后对话框会自动关闭

## 提示

- 仅支持上传 ZIP 格式文件
- ZIP 文件根目录必须包含 SKILL.md 文件
- 在包中包含所有必要的脚本和资源文件
