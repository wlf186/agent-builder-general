# 对话卡片

显示单个对话预览的卡片，支持选择、重命名和删除操作。

## 使用方法

1. 点击卡片选择并打开对话
2. 点击铅笔图标重命名对话
3. 点击垃圾桶图标删除对话
4. 在提示时确认删除

## 提示

- 当前对话会高亮显示
- 预览显示最后一条消息摘要
