# 对话历史

在滑出抽屉中访问对话历史。搜索、重命名和管理与智能体的历史对话。

## 使用方法

1. 点击聊天界面的历史图标打开抽屉
2. 使用搜索框查找特定对话
3. 点击对话以恢复它
4. 悬停在对话上可重命名或删除
5. 点击"新对话"开始新会话

## 提示

- 对话按智能体分组
- 搜索支持标题和消息内容
- 删除对话是不可恢复的
