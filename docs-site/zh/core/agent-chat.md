# 聊天界面

与AI智能体对话的主要区域。输入消息、附加文件，并观看打字机效果的实时响应。

## 使用方法

1. 在底部的文本输入框中输入消息
2. 可选：使用回形针图标附加文件（最多3个文件，每个100MB）
3. 按回车键或点击发送按钮提交消息
4. 观看AI的实时流式响应
5. 在可展开区域查看思考过程和工具调用

## 提示

- 使用 Shift+Enter 输入多行消息
- 点击工具调用查看智能体的操作
- 性能指标显示响应时间和Token使用量
