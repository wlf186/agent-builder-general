# 知识库管理

创建和管理智能体的知识库。 知识库存储智能体可以搜索的文档，以获取相关信息。

## 使用方法

1. 点击"新建知识库"创建新的知识库
2. 输入名称和可选描述
3. 点击"保存"创建知识库
4. 向知识库上传文档以进行索引

## 提示

- 使用描述性名称以便轻松识别知识库
- 将相关文档分组到同一知识库
- 支持的格式：PDF、DOCX、TXT、MD
