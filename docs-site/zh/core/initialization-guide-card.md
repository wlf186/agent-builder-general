# 初始化引导卡片

在智能体环境初始化期间显示，提供进度条、预估时间和可操作的配置项。

## 使用方法

1. 观察进度条了解初始化进度 2. 等待期间可配置可用选项（人设、MCP 服务） 3. 等待初始化完成后开始对话

## 提示

- 配置项在初始化过程中逐步变为可用 - 可以在等待期间设置人设和 MCP 服务
