# 知识库详情面板

管理知识库文档，支持拖拽上传、文档列表查看和语义检索测试。

## 使用方法

1. 拖拽文件或点击上传文档（支持 PDF、DOCX、TXT、MD）
2. 查看文档列表及状态（就绪、处理中、失败）
3. 输入查询词测试语义检索效果
4. 点击删除图标移除文档

## 提示

- 文档会自动分块并建立向量索引
- 使用检索测试功能验证文档质量，确保聊天时能正确召回
