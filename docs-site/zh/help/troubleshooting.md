# 故障排除

本指南帮助您解决 Agent Builder 的常见问题。

## 智能体问题

### 智能体不响应

**症状：** 智能体不回复或似乎卡住了。

**解决方案：**
1. **检查网络连接** - Agent Builder 需要稳定的连接
2. **刷新页面** - 有时连接会断开，需要重新建立
3. **检查智能体状态** - 在侧边栏中查看绿色指示器
4. **尝试其他智能体** - 如果其他智能体正常工作，问题可能只出现在某个智能体上

### 智能体给出错误答案

**症状：** 智能体提供不正确或不相关的信息。

**解决方案：**
1. **优化人设** - 使智能体的指令更加具体
2. **检查知识库** - 确保上传了正确的文档
3. **更具体地提问** - 在问题中提供更清晰的上下文
4. **检查启用的工具** - 确保相关工具已启用

### 智能体总是使用错误的工具

**症状：** 智能体调用不必要的工具或忽略相关工具。

**解决方案：**
1. **审查人设** - 添加关于何时使用哪些工具的明确指令
2. **禁用未使用的工具** - 移除智能体不应使用的工具
3. **检查工具描述** - 描述不清晰的工具可能会让智能体困惑

## 知识库问题

### 文档无法被找到

**症状：** 即使文档存在，智能体也说"我没有关于那个的信息"。

**解决方案：**
1. **检查文档上传** - 验证文件是否成功上传（无错误消息）
2. **等待处理** - 大型文档可能需要几分钟来索引
3. **尝试不同的措辞** - AI 使用语义搜索，所以尝试同义词
4. **检查文件格式** - 某些 PDF 是基于图像的，无法读取（尝试使用 DOCX）

### 上传失败

**症状：** 上传文档时出现错误消息。

**解决方案：**
1. **检查文件大小** - 最大为每个文件 100MB
2. **检查文件格式** - 支持：PDF、DOCX、TXT、MD
3. **尝试较小的批次** - 一次上传较少的文件
4. **检查网络** - 不稳定的连接可能导致上传失败

### 搜索结果不相关

**症状：** 智能体从知识库检索错误的信息。

**解决方案：**
1. **使用更具体的查询** - 添加上下文帮助 AI 理解
2. **检查文档内容** - 确保信息确实存在于您的文档中
3. **清理文档** - 删除过时或冲突的信息

## MCP 服务问题

### 服务显示为"已断开"

**症状：** MCP 服务状态显示红色或"已断开"。

**解决方案：**
1. **关闭并重新开启服务** - 有时会自动重新连接
2. **检查服务器状态** - 联系您的管理员验证服务是否正在运行
3. **检查服务配置** - 确保服务 URL 正确

### 工具调用失败

**症状：** 智能体尝试使用工具但收到错误。

**解决方案：**
1. **检查服务日志** - 联系您的管理员获取详细的错误日志
2. **验证工具参数** - 某些工具需要特定的输入格式
3. **尝试更简单的查询** - 复杂的请求可能会超时

## 性能问题

### 响应缓慢

**症状：** 智能体需要很长时间才能响应。

**解决方案：**
1. **减少知识库大小** - 太多文档会减慢搜索
2. **禁用未使用的工具** - 每个工具都会增加处理开销
3. **简化查询** - 将复杂问题分解为更小的部分
4. **检查服务器负载** - 高峰时段响应时间可能较慢

### 流式响应中途停止

**症状：** 响应在完成前中断。

**解决方案：**
1. **刷新页面** - 连接可能已断开
2. **检查网络稳定性** - 不稳定的连接会中断流式传输
3. **重试** - 临时服务器问题可能会自行解决

## 界面问题

### 侧边栏不加载

**症状：** 智能体列表或对话历史不显示。

**解决方案：**
1. **硬刷新** - 按 Ctrl+Shift+R（Mac 上为 Cmd+Shift+R）
2. **清除浏览器缓存** - 旧的缓存数据可能导致问题
3. **尝试其他浏览器** - 推荐使用 Chrome 或 Firefox

### 聊天输入不工作

**症状：** 无法输入或发送消息。

**解决方案：**
1. **点击输入框** - 它可能没有获得焦点
2. **检查错误消息** - 查找红色横幅或通知
3. **刷新页面** - UI 状态可能已损坏

## 错误消息

### "环境初始化失败"

这意味着智能体的执行环境无法启动。

**解决方案：**
1. 等待几分钟后重试
2. 联系您的管理员 - 这是服务器端问题

### "速率限制超出"

请求次数过多。

**解决方案：**
1. 等待几分钟后继续
2. 减少消息频率
3. 联系您的管理员增加限制

### "Token 限制超出"

您的消息或响应太长。

**解决方案：**
1. 缩短您的消息
2. 将请求分成多条消息
3. 要求智能体更简洁

## 仍有问题？

1. **查看 [常见问题](./faq)** - 您的问题可能已有答案
2. **联系您的管理员** - 他们可以检查服务器日志和配置
3. **报告错误** - 如果您认为这是软件错误，请报告并提供：
   - 您尝试做什么
   - 实际发生了什么
   - 您看到的任何错误消息
   - 如果可能，提供截图
