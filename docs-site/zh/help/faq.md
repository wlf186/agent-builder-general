# 常见问题

## 一般问题

### 什么是 Agent Builder？

Agent Builder 是一个创建 AI 智能助手（智能体）的平台。智能体可以与您聊天、使用工具、搜索文档，并与其他智能体协作。

### 我需要会编程吗？

不需要！Agent Builder 专为非技术用户设计。您可以使用自然语言描述来创建智能体。

### 我的 AI 智能体能做什么？

您的智能体可以：
- 用自然语言与您对话
- 使用工具（计算器、网络搜索等）
- 搜索上传的文档（知识库）
- 将任务委派给其他专业智能体

## 智能体

### 如何创建新智能体？

1. 点击左侧边栏的 **"创建智能体"** 按钮
2. 输入名称和描述
3. 编写智能体的人设（它应该做什么）
4. 点击 **保存**

### 什么是"人设"？

人设是描述智能体应如何行为的说明。它告诉 AI：
- 它扮演什么角色（例如："你是一个有用的助手"）
- 它应该处理什么任务
- 使用什么语气

### 创建智能体后可以修改吗？

可以！点击侧边栏中的智能体，然后点击设置图标来修改其人设、工具或知识库。

### 我可以创建多少个智能体？

没有硬性限制，但我们建议保持数量在可控范围内（5-10个）以便更好地组织。

## 知识库

### 支持哪些文件类型？

- **文档**：PDF、DOCX、TXT、MD
- **最大文件大小**：每个文件 100MB
- **每次最多上传**：10个文件

### AI 如何搜索我的文档？

AI 使用语义搜索来查找相关信息。它理解您查询的含义，而不仅仅是关键词匹配。

### 可以更新知识库中的文档吗？

可以！打开知识库并上传新文档或删除现有文档。更改会立即生效。

### 我的数据安全吗？

您的文档被安全存储，只能通过您的智能体访问。有关具体安全策略，请联系您的管理员。

## 对话

### 可以查看对话历史吗？

可以！点击左侧的对话抽屉（历史记录图标）查看所有过去的对话。

### 如何删除对话？

1. 打开对话抽屉
2. 找到要删除的对话
3. 点击删除图标
4. 确认删除

### 可以重命名对话吗？

目前，对话会根据其内容自动命名。尚不支持手动重命名。

## 工具与 MCP 服务

### 什么是 MCP 服务？

MCP（模型上下文协议）服务是智能体可以使用的外部工具，例如：
- 计算器
- 网络搜索
- 数据库查询
- 自定义集成

### 如何为智能体启用工具？

1. 打开智能体的设置
2. 进入 **MCP 服务** 部分
3. 开启您想启用的服务
4. 保存更改

### 工具不工作怎么办？

1. 检查智能体是否启用了该服务
2. 尝试关闭并重新启用服务
3. 查看 [故障排除](./troubleshooting) 页面了解常见问题
4. 如果问题持续存在，请联系您的管理员

## 性能

### 为什么智能体响应很慢？

以下几个因素可能影响响应时间：
- 需要多次工具调用的复杂查询
- 大型知识库搜索
- 服务器负载

### 性能指标是什么意思？

- **响应时间**：AI 生成响应所需的时间
- **Tokens**：处理的词/字符数量（影响成本）
- **工具调用**：使用了多少外部工具
