# Initialization Guide Card

Displays during agent environment initialization with progress bar, estimated time, and actionable configuration items.

## How to Use

1. Watch the progress bar as the environment initializes 2. While waiting, configure available options (persona, MCP services) 3. Wait for initialization to complete before starting chat

## Tips

- Configuration items become available progressively during initialization - You can set up persona and MCP services while waiting
