# Frequently Asked Questions

## General

### What is Agent Builder?

Agent Builder is a platform for creating AI assistants (agents) that can chat with you, use tools, search documents, and work with other agents as a team.

### Do I need to know how to code?

No! Agent Builder is designed for non-technical users. You can create agents using natural language descriptions.

### What can my AI agent do?

Your agent can:
- Chat with you in natural language
- Use tools (calculators, web searches, etc.)
- Search through uploaded documents (Knowledge Base)
- Delegate tasks to other specialized agents

## Agents

### How do I create a new agent?

1. Click the **"Create Agent"** button on the left sidebar
2. Enter a name and description
3. Write the agent's persona (what it should do)
4. Click **Save**

### What is a "persona"?

The persona is a description of how your agent should behave. It tells the AI:
- What role it plays (e.g., "You are a helpful assistant")
- What tasks it should handle
- What tone to use

### Can I edit an agent after creating it?

Yes! Click on the agent in the sidebar, then click the settings icon to modify its persona, tools, or knowledge bases.

### How many agents can I create?

There is no hard limit, but we recommend keeping the number manageable (5-10) for better organization.

## Knowledge Bases

### What file types are supported?

- **Documents**: PDF, DOCX, TXT, MD
- **Maximum file size**: 100MB per file
- **Maximum files per upload**: 10 files

### How does the AI search my documents?

The AI uses semantic search to find relevant information. It understands the meaning of your query, not just keyword matching.

### Can I update documents in a knowledge base?

Yes! Open the knowledge base and upload new documents or delete existing ones. Changes take effect immediately.

### Is my data secure?

Your documents are stored securely and only accessible through your agents. Contact your administrator for specific security policies.

## Conversations

### Can I see my conversation history?

Yes! Click the conversation drawer (history icon) on the left to see all past conversations.

### How do I delete a conversation?

1. Open the conversation drawer
2. Find the conversation you want to delete
3. Click the delete icon
4. Confirm the deletion

### Can I rename a conversation?

Currently, conversations are automatically named based on their content. Manual renaming is not yet supported.

## Tools & MCP Services

### What are MCP Services?

MCP (Model Context Protocol) services are external tools that your agent can use, such as:
- Calculators
- Web search
- Database queries
- Custom integrations

### How do I enable a tool for my agent?

1. Open your agent's settings
2. Go to the **MCP Services** section
3. Toggle on the services you want to enable
4. Save your changes

### A tool isn't working. What should I do?

1. Check that the service is enabled for your agent
2. Try disabling and re-enabling the service
3. Check the [Troubleshooting](./troubleshooting) page for common issues
4. Contact your administrator if the problem persists

## Performance

### Why is my agent slow?

Several factors can affect response time:
- Complex queries requiring multiple tool calls
- Large knowledge base searches
- Server load

### What do the performance metrics mean?

- **Response Time**: How long the AI took to generate a response
- **Tokens**: The number of words/characters processed (affects cost)
- **Tool Calls**: How many external tools were used

## Still have questions?

Contact your administrator or check the [Troubleshooting](./troubleshooting) page for more help.
