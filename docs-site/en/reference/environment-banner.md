# Environment Status Banner

Displays the skill execution environment initialization status at the top of the page. Shows a loading indicator during initialization and an error message with retry button if initialization fails.

## Tips

- The banner automatically disappears when the environment is ready
- Click the retry button to attempt environment initialization again
- Skill features are unavailable during initialization (10-30 seconds)
