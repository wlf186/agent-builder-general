# Sub-Agent Call Card

Displays real-time status, results, and statistics of sub-agent calls. Shows each call with status indicators (running, completed, timeout, failed), duration, token usage, and summary statistics.

## Tips

- Status colors indicate call progress: blue (running), green (completed), orange (timeout), red (failed)
- Token usage and duration are displayed when available
- Summary shows total duration and success rate
