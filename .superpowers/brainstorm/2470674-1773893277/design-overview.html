Key Design Decisions

  • KB in Sidebar: Same level as Agents and MCP - first-class citizen
  • One KB per Agent: Simpler mental model (can expand later)
  • Visual Feedback: "Searching..." indicator + source citations
  • Hybrid RAG: Vector search + BM25 keyword matching
