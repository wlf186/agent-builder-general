RAG Behavior

  1. Query Analysis: LLM determines if KB is relevant (via system prompt)
  2. Visual Feedback: "Searching HR Documents..." indicator
  3. Hybrid Retrieval: Vector search + BM25 keyword matching
  4. Context Injection: Retrieved chunks added to prompt (guaranteed usage)
  5. Source Citations: Clickable links showing document + page
🔑 Key Fix: Instead of relying on the LLM to choose to call rag_retrieve, we auto-inject retrieved context into the system prompt. This guarantees RAG is used when relevant.
