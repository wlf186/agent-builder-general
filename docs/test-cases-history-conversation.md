# 历史会话记录功能 - 测试用例文档

## 文档信息
- **版本**: v1.0
- **创建日期**: 2026-03-10
- **测试智能体**: `test001`
- **测试类型**: Playwright自动化测试 + 手动验证

---

## 1. 测试环境准备

### 1.1 前置条件
1. 后端服务已启动 (`python backend.py`，端口 20881)
2. 前端服务已启动 (`npm run dev`，端口 20880)
3. `test001` 智能体已配置且可用
4. MCP服务（如cold-jokes）已启用（用于测试工具调用）
5. Playwright已安装并配置

### 1.2 测试数据准备
每个会话至少包含2-3轮对话，测试对话内容如下：

| 会话 | 第1轮 | 第2轮 | 第3轮 |
|------|-------|-------|-------|
| 会话A | "讲一个冷笑话" | "再来10个类似的" | "再来5个风格不一样的" |
| 会话B | "你好，介绍一下你自己" | "你有哪些能力？" | "你能帮我做什么？" |
| 会话C | "今天天气怎么样？" | "北京呢？" | "明天会下雨吗？" |

### 1.3 测试环境启动命令
```bash
# 启动后端
cd /work/agent-builder-general
python backend.py &

# 启动前端
cd /work/agent-builder-general/frontend
npm run dev &

# 安装Playwright（如未安装）
cd /work/agent-builder-general/frontend
npm install playwright
npx playwright install chromium

# 运行测试
npx playwright test tests/history-conversation.spec.ts
```
