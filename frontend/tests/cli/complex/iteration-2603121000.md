### TC001: PDF 文件上传 API 测试

## 步骤

1. 创建一个最小有效 PDF 文件
2. 通过 API 上传:
   ```bash
   curl -X POST http://localhost:20881/api/agents/file-skill-tester/files \
     -F "file=@test-api.pdf;type=application/pdf"
   ```
3. 验证响应: `success: true`, `file.file_id` 存在, `file.filename` 为 `test-api.pdf`
4. 记录 `file_id`
5. 获取文件列表: `GET http://localhost:20881/api/agents/file-skill-tester/files`
6. 验证上传的文件在列表中
7. 删除文件: `DELETE http://localhost:20881/api/agents/file-skill-tester/files/{file_id}`
8. 再次获取文件列表，验证文件已删除

## 验证
- 上传返回成功，包含 file_id
- 文件列表中可见
- 删除后不再可见
