### TC2: 文件上传 API 测试

## 步骤

1. 创建一个最小有效 PDF 文件的二进制内容
2. 通过 API 上传 PDF 文件:
   ```bash
   curl -X POST http://localhost:20881/api/agents/test-iteration-2603111255/files \
     -F "file=@test-api.pdf;type=application/pdf"
   ```
3. 验证上传响应: `success` 为 true, `file.file_id` 存在, `file.filename` 为 `test-api.pdf`
4. 记录返回的 `file_id`
5. 通过 API 获取文件列表: `GET http://localhost:20881/api/agents/test-iteration-2603111255/files`
6. 验证上传的文件在列表中
7. 通过 API 删除文件: `DELETE http://localhost:20881/api/agents/test-iteration-2603111255/files/{file_id}`
8. 再次获取文件列表，验证文件已被删除

## 验证
- 上传返回 `success: true`，包含 `file_id`
- 文件列表中包含刚上传的文件
- 删除后文件列表中不再包含该文件
