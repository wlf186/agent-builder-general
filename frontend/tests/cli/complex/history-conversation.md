### TC-HC-003: 历史会话列表按时间排序

## 步骤

> 注意: 此步骤需要依次创建 3 个会话，每个会话发送 3 条消息

1. **创建会话 A (冷笑话测试)**:
   - 如果有 "新建" 按钮，点击新建
   - 等待 0.5 秒
   - 找到消息输入框，输入 "讲一个冷笑话"，按回车
   - 等待流式输出完成 (检查页面不再有 "生成中" 文本，输入框可用)
   - 等待 0.5 秒
   - 输入 "再来10个类似的"，按回车，等待完成
   - 等待 0.5 秒
   - 输入 "再来5个风格不一样的"，按回车，等待完成
   - 等待 2 秒

2. **创建会话 B (自我介绍测试)**:
   - 点击 "新建" 按钮，等待 0.5 秒
   - 依次发送:
     - "你好，介绍一下你自己" (等待完成 + 0.5秒)
     - "你有哪些能力？" (等待完成 + 0.5秒)
     - "你能帮我做什么？" (等待完成 + 0.5秒)
   - 等待 2 秒

3. **创建会话 C (天气查询测试)**:
   - 点击 "新建" 按钮，等待 0.5 秒
   - 依次发送:
     - "今天天气怎么样？" (等待完成 + 0.5秒)
     - "北京呢？" (等待完成 + 0.5秒)
     - "明天会下雨吗？" (等待完成 + 0.5秒)

4. **打开历史抽屉**:
   - 点击 "历史" 按钮
   - 在 snapshot 中统计会话卡片数量，验证至少 3 个
5. 关闭历史抽屉

## 验证
- 3 个会话成功创建
- 历史会话列表中至少有 3 个会话卡片
- 会话按时间从新到旧排序
