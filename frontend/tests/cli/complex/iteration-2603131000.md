### TC-12: 性能测试 - 并发创建多个智能体

## 步骤

使用 `run-code` 并发创建 3 个智能体:

```javascript
const agentNames = ['test-concurrent-1', 'test-concurrent-2', 'test-concurrent-3'];
const results = [];

await Promise.all(agentNames.map(async (name) => {
  const start = Date.now();
  try {
    const res = await fetch('http://localhost:20881/api/agents', {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify({ name, description: '并发测试' })
    });
    const elapsed = Date.now() - start;
    results.push({ name, elapsed, ok: res.ok });
  } catch (e) {
    results.push({ name, elapsed: Date.now() - start, ok: false });
  }
}));

console.log(results);
// 清理
for (const r of results) {
  await fetch(`http://localhost:20881/api/agents/${r.name}`, { method: 'DELETE' });
}
```

## 验证
- 所有 3 个创建请求都成功
- 每个创建响应时间 < 500ms
