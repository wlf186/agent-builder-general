## 2. 测试用例

### TC1: 前端文件上传组件测试

**测试目标**: 验证 FileUploader 组件功能正常

| 项目 | 内容 |
|------|------|
| 用例ID | TC1 |

**预期结果**:

- [ ] 回形针图标可点击

- [ ] 拖拽时显示蓝色边框

### TC2: 文件上传 API 测试

**测试目标**: 验证后端文件上传 API 功能正常

| 项目 | 内容 |
|------|------|
| 用例ID | TC2 |
| 优先级 | P0 |
| 前置条件 | 后端服务启动，存在测试 Agent |

**测试步骤**:

```bash
# 1. 上传 PDF 文件
curl -X POST "http://localhost:20881/api/agents/test001/files" \
  -F "file=@test.pdf"

# 2. 获取文件列表
curl "http://localhost:20881/api/agents/test001/files"

# 3. 下载文件
curl "http://localhost:20881/api/agents/test001/files/{file_id}/download" -o downloaded.pdf

# 4. 删除文件
curl -X DELETE "http://localhost:20881/api/agents/test001/files/{file_id}"
```

**预期结果**:

- [ ] 上传返回 `{"success": true, "file": {...}}`
- [ ] 文件列表包含上传的文件
- [ ] 下载文件与原文件一致
- [ ] 删除后文件列表不再包含该文件
