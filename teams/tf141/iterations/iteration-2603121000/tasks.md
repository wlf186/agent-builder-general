### T007: 为 AB-pdf 添加 main.py 入口脚本

**责任人**: be
**优先级**: P0
**预计工时**: 2h

**任务描述**:
为 AB-pdf Skill 创建统一的命令行入口脚本，支持以下功能：
- `extract_text`: 提取 PDF 文本内容
- `extract_forms`: 提取表单字段信息
- `fill_form`: 填充表单字段
- `convert_images`: 转换为图片

**验收标准**:
1. 脚本位于 `skills/builtin/AB-pdf/scripts/main.py`
2. 支持 `--help` 参数显示使用说明
3. 所有操作返回标准 JSON 格式
4. 错误情况返回明确的错误信息

**技术要点**:
- 参考 `系统应用架构文档.md` 第 5.4 节的脚本规范
- 使用 `argparse` 处理命令行参数
- 输出格式：`{"status": "success/error", "output": "...", "error": "..."}`
