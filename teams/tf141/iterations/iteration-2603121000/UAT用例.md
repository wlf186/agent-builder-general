## 一、UAT 概述

### 1.1 测试目标

验证文件上传与 Skill 执行功能的端到端可用性，确保用户能够：
1. 上传 PDF/DOCX 文件并获取 file_id
2. Agent 能正确识别上传的文件
3. Agent 能自动调用对应的 Skill 处理文件
4. 整个流程对用户透明，体验流畅

| 项目 | 配置 |
|------|------|
| 前端地址 | http://localhost:20880 |
| 后端地址 | http://localhost:20881 |
| 浏览器 | Chrome 90+ / Playwright |
| 测试智能体 | test001, skill-test-pdf, skill-test-doc |
| 模型服务 | TESTLLM |

### 1.3 测试数据
