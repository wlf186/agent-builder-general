## 二、功能测试用例

### TC001: PDF 文件上传 API 测试

**测试目标**: 验证 PDF 文件上传 API 功能正常

**前置条件**:
- 后端服务运行正常
- 测试智能体已创建

**测试步骤**:
1. 构造有效的 PDF 文件内容
2. 调用 `POST /api/agents/{agent_name}/files` 上传文件
3. 验证返回的 file_id 有效
4. 调用 `GET /api/agents/{agent_name}/files` 验证文件在列表中
5. 调用 `DELETE /api/agents/{agent_name}/files/{file_id}` 删除文件
6. 验证删除后文件不在列表中

**预期结果**:
- 上传成功返回 `{"success": true, "file": {"file_id": "...", "filename": "..."}}`
- 文件列表包含上传的文件
- 删除后文件列表不包含该文件

**优先级**: P0
